Descripción detallada

Módulo que permite realizar operaciones unarias y binarias con matrices de booleanos.

La sintaxis de llamada es: calcular OPERACION [Parámetros]

  • OPERACION: AND, OR (binarias); NOT, TRS (unarias)
  • [Parámetros]: nombre(s) de fichero(s) donde se encuentra(n) la(s) matrizo matrices en cada caso. Si no se especifican, se leerá desde la entrada estándar.

Un ejemplo de ejecución es:

> calcular NOT identidad.mat (Identidad de orden 4)
4 4
0 1 1 1
1 0 1 1
1 1 0 1
1 1 1 0
